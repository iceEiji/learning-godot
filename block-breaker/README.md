# ブロック崩し

## ディレクトリ構成

```
res://
├── screens/          # 画面単位
├── objects/          # ゲーム固有の登場物
├── components/       # 再利用可能なUI部品（Webでいうcomponents）
├── systems/          # ゲームロジック（AutoLoadに登録するもの）
├── data/             # リソースデータ
│   ├── stages/       # ステージ管理
│   └── config/       # 定数など設定ファイル
├── assets/
│   ├── sprites/      # 画像
│   ├── sounds/       # 音楽・サウンドエフェクト
│   └── fonts/        # フォント
└── shared/           # 汎用ユーティリティ
```
